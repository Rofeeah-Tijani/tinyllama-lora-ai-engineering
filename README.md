# TinyLlama LoRA Fine-Tuning Project

This project fine-tunes TinyLlama-1.1B-Chat using LoRA on a custom AI Engineering instruction dataset.

## Model
- Base Model: TinyLlama/TinyLlama-1.1B-Chat-v1.0
- Fine-Tuning Method: LoRA
- Dataset Size: 1000 instruction-response examples

## Features
- AI Engineering concepts
- Instruction-following responses
- Lightweight LoRA adapter (~4.5 MB)

## Deployment
Built with Gradio and Hugging Face Spaces.
